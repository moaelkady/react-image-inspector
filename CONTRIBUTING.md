# Contributing

Thanks for contributing to `react-image-inspector`.

## Install

```bash
npm install
```

## Run development demo

```bash
npm run dev
```

## Build package

```bash
npm run build
```

## Typecheck

```bash
npm run typecheck
```

## Pull request expectations

- Keep changes focused and reviewable.
- Include docs updates for new props or feature flags.
- Preserve accessibility behavior for controls/keyboard usage.
- Avoid breaking existing public API names unless absolutely required.

## Dependency policy

- Keep dependencies minimal.
- Do not add heavy runtime packages without strong justification.
- Prefer small internal utilities over new external packages.
